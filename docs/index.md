## 为什么选择MkDocs Material？

- **美观专业**: 基于Google Material Design，提供现代化的UI体验
- **响应式设计**: 完美适配各种设备和屏幕尺寸
- **功能丰富**: 支持代码高亮、图表、数学公式等高级功能
- **性能优异**: 优化的加载速度和SEO友好
- **高度可定制**: 通过CSS和配置完全控制外观和行为
- **活跃社区**: 持续更新和支持

## 快速开始

1. **安装MkDocs**:
   ```bash
   pip install mkdocs mkdocs-material
   ```

2. **创建项目**:
   ```bash
   mkdocs new my-docs
   cd my-docs
   ```

3. **启动开发服务器**:
   ```bash
   mkdocs serve
   ```

4. **访问文档**:
   打开浏览器访问 [http://localhost:8000](http://localhost:8000)
